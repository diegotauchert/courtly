## Run using Docker

```bash
make deploy_dev
```

and then 

```bash
make dev
```

Open the link [http://localhost:3006](http://localhost:3006)

## or run using Node

`Node Version: <= 18`

## Installation

First, install the dependencies

```bash
npm install
```

and then 

```bash
npm run dev
```

Open on browser [http://localhost:3000](http://localhost:3000) .
